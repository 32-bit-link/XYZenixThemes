Source | Download | Addons | DTM-16 V5 (unfinished) DTM-16 (Discord Time Machine 2016) is a theme that reverts most of the UI changes that occurred after 2016.

Notes

You can use these BetterDiscord plugins to make DTM-16 feel closer to old Discord: OnlineFriendCount, ColorSighted
Light mode requires enabling Dark Sidebar option. If you want full light mode, use the Full Light Mode addon.
The theme should work properly using other client mods such as Powercord. I've only used BetterDiscord and Powercord, so I can't guarantee it works flawlessly on other ones.
DTM-16 V5 IS NOT ENTIRELY FINISHED SO I RECOMMEND USING V4. I plan to make V5 fully light mode compatible so it should support both full light mode and dark sidebar in the future.
